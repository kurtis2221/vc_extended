# GTA Vice City Extended

## Info
Adds extra functions to the game
- Save everywhere
- Select weapon slot by key
- Change radio station (unstable)
- When wanted give up key
- Change vehicle color in construction site

## Tools Needed To load the project properly
- Visual C# 2008 Express or newer
